
Tic-Tac-Toe Game in C
A simple and interactive Tic-Tac-Toe game built using the C programming language.
This project is beginner-friendly and helps understand arrays, functions, loops, and game logic in C

 Features:
Two-player gameplay (Player 1 → X, Player 2 → O)
3×3 grid displayed in the terminal
Validates player moves
Prevents overwriting a filled spot

Automatically checks for:
Win
Draw
Invalid move

 How the Game Works
1. The board is stored as a 3×3 array.
2. Players choose positions from 1 to 9.
3. After every move:

The board updates

The game checks if someone has won
4. If all positions are filled → Match Draw
